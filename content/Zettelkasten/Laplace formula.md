# Laplace formula
Let $D$ be some matrix. Laplace formula says: 

For any row $i$  
$$\det(D) = \sum_j (-1)^{i+j}d_{ij}\det(M_{ij})$$
For any column $j$
$$\det(D) = \sum_i (-1)^{i+j}d_{ij}\det(M_{ij})$$
